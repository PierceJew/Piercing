# Telegram-каталог прикрас для пірсингу

Простий бот без Mini App і без Google Таблиць.

## Що всередині

- Головне меню: `Каталог` і `Зв'язатися`
- Каталог: частина тіла -> вид проколу -> прикраси
- Адмін-панель: додавання розділів, проколів і прикрас, редагування фото, ціни та опису
- Товари зберігаються в `catalog.json`
- Токен, контакт менеджера та адміни зберігаються в `.env`

## Запуск

1. Встановити залежності:

```powershell
pip install -r requirements.txt
```

2. Створити `.env` поруч із `bot.py`:

```powershell
copy .env.example .env
```

3. Заповнити `.env`:

```env
BOT_TOKEN=ваш_токен
MANAGER_URL=https://t.me/username
ADMIN_IDS=ваш_telegram_id
```

4. Запустити:

```powershell
python bot.py
```

## Адмін-панель

1. Запустіть бота.
2. Напишіть боту `/whoami`.
3. Скопіюйте свій Telegram ID у `.env`:

```env
ADMIN_IDS=123456789
```

Якщо адмінів кілька, вкажіть ID через кому:

```env
ADMIN_IDS=123456789,987654321
```

Після цього перезапустіть бота і напишіть `/admin`.

В адмінці можна:

- додати частину тіла;
- додати вид проколу;
- додати прикрасу з фото;
- змінити назву, фото, матеріал, розмір, ціну та опис прикраси;
- видалити прикрасу.

## Як додавати фото

Через адмінку просто надішліть фото боту на відповідному кроці.

Якщо додаєте вручну через `catalog.json`, створіть папку `images` поруч із `bot.py` і вкажіть шлях:

```json
"image": "images/helix-clicker.jpg"
```

Якщо фото поки немає, залиште поле порожнім:

```json
"image": ""
```
